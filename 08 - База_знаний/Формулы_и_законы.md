# Основные формулы и законы термодинамики

## 📐 Уравнения состояния

### Идеальный газ
$$pV = mRT$$
$$pv = RT$$
$$p = \rho RT$$

### Реальный газ (Ван-дер-Ваальс)
$$\left(p + \frac{a}{v^2}\right)(v - b) = RT$$

## 🔄 Первый закон термодинамики

### Общая форма
$$Q = \Delta U + W$$
$$\delta Q = dU + \delta W$$

### Для закрытой системы
$$\delta Q = dU + p dV$$

### Для открытой системы
$$\delta Q = dH + \delta W_{тех}$$

### Удельные величины
$$\delta q = du + \delta w$$
$$q = \Delta u + w$$

## 🌡️ Теплоёмкость

### Определения
$$c = \frac{\delta Q}{dT}$$
$$c_v = \left(\frac{\partial u}{\partial T}\right)_v$$
$$c_p = \left(\frac{\partial h}{\partial T}\right)_p$$

### Для идеального газа
$$c_p - c_v = R$$
$$k = \frac{c_p}{c_v}$$

## 📊 Термодинамические процессы

### Политропный процесс ($pv^n = const$)

**Связь параметров:**
$$\frac{T_2}{T_1} = \left(\frac{v_1}{v_2}\right)^{n-1} = \left(\frac{p_2}{p_1}\right)^{\frac{n-1}{n}}$$
$$\frac{p_2}{p_1} = \left(\frac{v_1}{v_2}\right)^n$$

**Работа:**
$$l = \frac{R}{n-1}(T_1 - T_2) = \frac{p_1v_1 - p_2v_2}{n-1}$$

**Теплота:**
$$q = c_n (T_2 - T_1)$$
$$c_n = c_v \frac{n-k}{n-1}$$

### Адиабатный процесс ($s = const$, $n = k$)

**Уравнение Пуассона:**
$$pv^k = const$$
$$\frac{T_2}{T_1} = \left(\frac{v_1}{v_2}\right)^{k-1} = \left(\frac{p_2}{p_1}\right)^{\frac{k-1}{k}}$$

**Работа:**
$$l = \frac{R}{k-1}(T_1 - T_2) = \frac{p_1v_1 - p_2v_2}{k-1}$$

### Изотермический процесс ($T = const$, $n = 1$)

$$pv = const$$
$$l = RT \ln\frac{v_2}{v_1} = RT \ln\frac{p_1}{p_2}$$
$$q = l$$

### Изобарный процесс ($p = const$, $n = 0$)

$$l = p(v_2 - v_1) = R(T_2 - T_1)$$
$$q = c_p(T_2 - T_1) = \Delta h$$

### Изохорный процесс ($v = const$, $n = \infty$)

$$l = 0$$
$$q = c_v(T_2 - T_1) = \Delta u$$

## 🔄 Второй закон термодинамики

### Энтропия
$$dS = \frac{\delta Q_{обр}}{T}$$
$$\Delta S = \int_1^2 \frac{\delta Q_{обр}}{T}$$

### Для идеального газа
$$\Delta s = c_v \ln\frac{T_2}{T_1} + R \ln\frac{v_2}{v_1}$$
$$\Delta s = c_p \ln\frac{T_2}{T_1} - R \ln\frac{p_2}{p_1}$$
$$\Delta s = c_v \ln\frac{p_2}{p_1} + c_p \ln\frac{v_2}{v_1}$$

### Статистический смысл (Больцман)
$$S = k \ln W$$

## ⚙️ Циклы

### Цикл Карно
$$\eta_t^{Карно} = 1 - \frac{T_2}{T_1}$$
$$\varepsilon^{Карно} = \frac{T_2}{T_1 - T_2}$$

### Цикл Отто
$$\eta_t = 1 - \frac{1}{\varepsilon^{k-1}}$$

### Цикл Дизеля
$$\eta_t = 1 - \frac{1}{\varepsilon^{k-1}} \cdot \frac{\rho^k - 1}{k(\rho - 1)}$$

### Цикл Тринклера
$$\eta_t = 1 - \frac{1}{\varepsilon^{k-1}} \cdot \frac{\lambda \rho^k - 1}{(\lambda - 1) + k\lambda(\rho - 1)}$$

## ❄️ Холодильные циклы

### Холодильный коэффициент
$$\varepsilon = \frac{q_0}{l} = \frac{i_1 - i_4}{i_2 - i_1}$$

### Холодопроизводительность
$$q_0 = i_1 - i_4$$

### Работа компрессора
$$l = i_2 - i_1$$

### Теплота конденсации
$$q_k = i_2 - i_3$$

## 📈 Соотношения Максвелла

$$\left(\frac{\partial T}{\partial v}\right)_s = -\left(\frac{\partial p}{\partial s}\right)_v$$
$$\left(\frac{\partial T}{\partial p}\right)_s = \left(\frac{\partial v}{\partial s}\right)_p$$
$$\left(\frac{\partial p}{\partial T}\right)_v = \left(\frac{\partial s}{\partial v}\right)_T$$
$$\left(\frac{\partial v}{\partial T}\right)_p = -\left(\frac{\partial s}{\partial p}\right)_T$$
