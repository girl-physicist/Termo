# Сравнение прямых и обратных термодинамических циклов

## 📊 Принципиальное отличие

### Прямой цикл (ДВС)
T↑ T↑  
│ q₁ │ q₁  
│ ↓ │ ↓  
│ ┌────┐ │ ┌────┐  
│ │ │ → L │ │ │ ← L  
│ └────┘ │ └────┘  
│ ↓ │ ↑  
│ q₂ │ q₂  
└─────→ s └─────→ s  
ДВС Холодильная машина

### Сравнение в таблице

| Параметр | Прямой цикл (ДВС) | Обратный цикл (ХМ) |
|----------|-------------------|---------------------|
| Цель | Получить работу | Перенести теплоту |
| Направление | По часовой стрелке | Против часовой |
| Знак работы | $L > 0$ (отдаётся) | $L < 0$ (затрачивается) |
| Характеристика | $\eta = L/q_1$ | $\varepsilon = q_2/L$ |
| Предельный цикл | [[Цикл_Карно]] прямой | [[Цикл_Карно]] обратный |

## 🔄 Общие элементы

### Компрессор
В обоих типах машин есть компрессор:
- В ДВС — компрессор (турбонаддув) или сам цилиндр при сжатии
- В ХМ — компрессор как основной элемент

См. [[Политропные_процессы]] — теория сжатия общая!

### Теплообменники
- ДВС: радиатор охлаждения, интеркулер
- ХМ: конденсатор, испаритель

Общая теория теплообмена: [[Теплопередача]]

## 🧮 Сквозные расчёты

### Один и тот же компрессор
Рассмотрим компрессор, работающий в разных режимах:

**В ДВС (турбонаддув):**
$$l_{сж} = \frac{RT_1}{n-1}\left[\left(\frac{p_2}{p_1}\right)^{\frac{n-1}{n}} - 1\right]$$

**В холодильной машине:**
$$l_{сж} = h_2 - h_1$$

Формулы разные, но физика одна!
