# Второй закон термодинамики

## 🎯 Что студент узнает из этого материала
- Формулировки второго закона термодинамики
- Понятие энтропии и её физический смысл
- Направление тепловых процессов
- Максимальный КПД тепловых машин (цикл Карно)
- Эксергия и потеря работоспособности

## 📝 Теоретическая часть

### Формулировки второго закона

**Формулировка Клаузиуса (о теплоте):**
> Теплота не может самопроизвольно переходить от более холодного тела к более нагретому.

**Формулировка Томсона (Кельвина):**
> Невозможно построить периодически действующую тепловую машину, единственным результатом которой было бы превращение теплоты в работу при охлаждении одного источника теплоты.

**Формулировка Планка:**
> Невозможно построить машину, которая полностью превращала бы теплоту в работу без каких-либо других изменений в природе.

**Формулировка через энтропию:**
> В изолированных системах энтропия либо возрастает (необратимые процессы), либо остаётся постоянной (обратимые процессы).

### Энтропия

**Определение:**
$$dS = \frac{\delta Q_{обр}}{T}$$

**Физический смысл:**
- Мера беспорядка в системе
- Мера рассеяния энергии
- Стрела времени (направление процессов)

**Для идеального газа:**
$$\Delta s = c_v \ln\frac{T_2}{T_1} + R \ln\frac{v_2}{v_1}$$
$$\Delta s = c_p \ln\frac{T_2}{T_1} - R \ln\frac{p_2}{p_1}$$
$$\Delta s = c_v \ln\frac{p_2}{p_1} + c_p \ln\frac{v_2}{v_1}$$

### T-s диаграмма
T  
↑  
│ ┌────┐  
│ │ │  
│ │ │  
│ │ │  
│ └────┘  
│ q = ∫T ds  
└────────────→ s

На T-s диаграмме:
- Площадь под кривой процесса = теплота процесса
- Изотерма — горизонтальная линия
- Адиабата — вертикальная линия
- Площадь цикла = работа цикла

## 🔄 Цикл Карно

### Описание цикла
Цикл Карно состоит из четырёх обратимых процессов:
T  
↑  
│ 1────2  
│ │ │  
│ │ │  
│ 4────3  
└──────────→ s

| Участок | Процесс | Характеристика |
|---------|---------|----------------|
| 1→2 | Изотермическое расширение | $T = T_1 = const$, подвод тепла |
| 2→3 | Адиабатное расширение | $s = const$, охлаждение до $T_2$ |
| 3→4 | Изотермическое сжатие | $T = T_2 = const$, отвод тепла |
| 4→1 | Адиабатное сжатие | $s = const$, нагрев до $T_1$ |

### Термический КПД цикла Карно
$$\eta_t^{Карно} = 1 - \frac{T_2}{T_1} = \frac{T_1 - T_2}{T_1}$$

**Важнейшие следствия:**
1. КПД не зависит от рабочего тела
2. КПД < 1 всегда (кроме $T_2 = 0$ К)
3. Для повышения КПД нужно увеличивать $T_1$ или уменьшать $T_2$
4. Ни один реальный цикл не может иметь КПД выше цикла Карно при тех же температурах

### Холодильный коэффициент цикла Карно
Для обратного цикла (холодильная машина):
$$\varepsilon^{Карно} = \frac{T_2}{T_1 - T_2}$$

Для теплового насоса:
$$\mu^{Карно} = \frac{T_1}{T_1 - T_2}$$

## 📊 Необратимость процессов

### Причины необратимости
1. **Трение** — превращение работы в теплоту
2. **Теплообмен при конечной разности температур**
3. **Дросселирование** (мятие газа)
4. **Диффузия** и смешение
5. **Химические реакции**
6. **Электрическое сопротивление**

### Рост энтропии в необратимых процессах

**Теплообмен при ΔT ≠ 0:**
$$\Delta S = Q\left(\frac{1}{T_2} - \frac{1}{T_1}\right) > 0$$

**Дросселирование (эффект Джоуля-Томсона):**
$$\Delta s = \int_{p_1}^{p_2} \left(\frac{\partial s}{\partial p}\right)_T dp > 0$$

**Смешение газов (парадокс Гиббса):**
$$\Delta S = -R \sum n_i \ln x_i$$

## ⚡ Эксергия

### Определение
**Эксергия** (доступная энергия, работоспособность) — максимальная работа, которую может совершить система при переходе в равновесие с окружающей средой.

**Анаргия** (связанная энергия) — часть энергии, которая не может быть превращена в работу.

$$E = U + p_0 V - T_0 S - G_0$$

Для теплового потока:
$$E_q = Q\left(1 - \frac{T_0}{T}\right)$$

### Потери эксергии (теорема Гюи-Стодолы)
$$D = T_0 \Delta S_{системы}$$

Потери эксергии пропорциональны росту энтропии и температуре окружающей среды.

## 🔧 Применение к ДВС и холодильным машинам

### Двигатель внутреннего сгорания
Реальный КПД всегда ниже КПД цикла Карно из-за:
- Необратимости сгорания (химическая реакция)
- Теплообмена при конечной разности температур
- Трения
- Гидравлических потерь

### Холодильная машина
Реальный холодильный коэффициент всегда ниже обратного цикла Карно:
$$\varepsilon_{реал} = \varepsilon_{Карно} \cdot \eta_{экс}$$

где $\eta_{экс}$ — эксергетический КПД (обычно 0.4-0.6)

## 🧮 Примеры расчётов

### Пример 1. КПД цикла Карно
**Условие:** Найти максимально возможный КПД теплового двигателя, работающего между температурами 800°C и 30°C.

**Решение:**
1. Переводим в Кельвины:
   $$T_1 = 800 + 273 = 1073 \text{ К}$$
   $$T_2 = 30 + 273 = 303 \text{ К}$$

2. КПД цикла Карно:
   $$\eta_{Карно} = 1 - \frac{T_2}{T_1} = 1 - \frac{303}{1073} = 1 - 0.282 = 0.718 = 71.8\%$$

**Вывод:** Даже в идеальном случае нельзя получить КПД выше 71.8%. Реальные ДВС имеют КПД 30-45%.

### Пример 2. Изменение энтропии при теплообмене
**Условие:** К горячему источнику с температурой 500 К подводится 1000 кДж теплоты и передаётся холодному источнику с температурой 300 К. Найти изменение энтропии системы.

**Решение:**
1. Изменение энтропии горячего источника:
   $$\Delta S_1 = -\frac{Q}{T_1} = -\frac{1000}{500} = -2 \text{ кДж/К}$$

2. Изменение энтропии холодного источника:
   $$\Delta S_2 = +\frac{Q}{T_2} = +\frac{1000}{300} = +3.333 \text{ кДж/К}$$

3. Суммарное изменение:
   $$\Delta S = \Delta S_1 + \Delta S_2 = -2 + 3.333 = 1.333 \text{ кДж/К} > 0$$

**Вывод:** Энтропия возросла, процесс необратим.

### Пример 3. Потеря работоспособности
**Условие:** В теплообменнике при разности температур 50 К передаётся 1000 кДж теплоты. Температура окружающей среды 290 К. Найти потерю эксергии.

**Решение:**
1. Средняя температура горячего теплоносителя 400 К, холодного 350 К
2. Изменение энтропии:
   $$\Delta S = Q\left(\frac{1}{T_хол} - \frac{1}{T_{гор}}\right) = 1000\left(\frac{1}{350} - \frac{1}{400}\right) = 1000(0.002857 - 0.0025) = 0.357 \text{ кДж/К}$$

3. Потеря эксергии (теорема Гюи-Стодолы):
   $$D = T_0 \Delta S = 290 \cdot 0.357 = 103.5 \text{ кДж}$$


## 🔄 Применение к разным разделам

### Прямые и обратные циклы
Второй закон устанавливает связь:

| Характеристика | ДВС (прямой цикл) | Холодильная машина (обратный цикл) |
|----------------|-------------------|-------------------------------------|
| Направление | Тепло → Работа | Работа → Холод |
| КПД/ε | $\eta = 1 - T_2/T_1$ | $\varepsilon = T_2/(T_1-T_2)$ |
| Предел | [[Цикл_Карно]] | [[Цикл_Карно]] (обратный) |
