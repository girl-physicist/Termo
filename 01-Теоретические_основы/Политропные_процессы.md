# Политропные процессы

## 📝 Теоретическая часть

### Определение
**Политропный процесс** — это термодинамический процесс, в котором теплоёмкость газа остаётся постоянной на протяжении всего процесса.

### Основное уравнение
$$p \cdot v^n = const$$

где:
- $p$ — давление
- $v$ — удельный объём
- $n$ — **показатель политропы** (постоянная величина для данного процесса)

### Соотношения между параметрами
Для политропного процесса справедливы следующие зависимости:

$$\frac{T_2}{T_1} = \left(\frac{v_1}{v_2}\right)^{n-1}$$

$$\frac{T_2}{T_1} = \left(\frac{p_2}{p_1}\right)^{\frac{n-1}{n}}$$

$$\frac{p_2}{p_1} = \left(\frac{v_1}{v_2}\right)^{n}$$

### Работа и теплота в политропном процессе

**Работа изменения объёма:**
$$l = \frac{R}{n-1}(T_1 - T_2) = \frac{p_1v_1 - p_2v_2}{n-1}$$

**Количество теплоты:**
$$q = c_n (T_2 - T_1)$$

где $c_n$ — политропная теплоёмкость:
$$c_n = c_v \frac{n-k}{n-1}$$

**Изменение внутренней энергии:**
$$\Delta u = c_v (T_2 - T_1)$$

## 📊 Частные случаи политропных процессов

| n | Название процесса | Уравнение | Характеристика |
|---|-------------------|-----------|----------------|
| 0 | Изобарный | $p = const$ | Давление постоянно |
| 1 | Изотермический | $T = const$ | Температура постоянна |
| k | Адиабатный | $s = const$ | Нет теплообмена |
| ∞ | Изохорный | $v = const$ | Объём постоянен |

### Визуализация частных случаев
p  
↑  
│ n=0 (изобара)  
│ n=1 (изотерма)  
│ n=k (адиабата)  
│ n=∞ (изохора)  
└─────────→ v

## 🔧 Практическое значение
Политропные процессы широко используются для описания реальных процессов в:
- Двигателях внутреннего сгорания (процессы сжатия и расширения)
- Компрессорах
- Детандерах
- Теплообменных аппаратах
