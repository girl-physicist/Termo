# Лабораторная работа №2
## Исследование термодинамических циклов ДВС (Отто и Дизеля)

## 🎯 Цель работы
1. Изучить теоретические основы циклов Отто и Дизеля
2. Научиться строить циклы в p-v и T-s координатах

4. Исследовать влияние основных параметров на КПД

## 📋 Теоретическое введение

### Цикл Отто (бензиновый двигатель)
- Подвод тепла при $v = const$
- Степень сжатия $\varepsilon = 6 \ldots 11$
- Рабочее тело — бензовоздушная смесь

### Цикл Дизеля (дизельный двигатель)
- Подвод тепла при $p = const$
- Степень сжатия $\varepsilon = 14 \ldots 22$
- Воспламенение от сжатия

#### Задания:
1. Рассчитайте параметры во всех характерных точках цикла
2. Определите подведённую и отведённую теплоту
3. Вычислите работу цикла и термический КПД
4. Постройте цикл в p-v координатах
5. Постройте цикл в T-s координатах

**Расчётные формулы:**
$$p_2 = p_1 \varepsilon^k$$
$$T_2 = T_1 \varepsilon^{k-1}$$
$$p_3 = p_2 \lambda$$
$$T_3 = T_2 \lambda$$
$$p_4 = p_3 \left(\frac{1}{\varepsilon}\right)^k = p_1 \lambda$$
$$T_4 = T_3 \left(\frac{1}{\varepsilon}\right)^{k-1} = T_1 \lambda$$

$$q_1 = c_v(T_3 - T_2)$$
$$q_2 = c_v(T_4 - T_1)$$
$$l = q_1 - q_2$$
$$\eta_t = 1 - \frac{1}{\varepsilon^{k-1}}$$

### Часть 2. Исследование цикла Дизеля

#### Варианты заданий

| 4 | 18 | 2.3 | 295 | 0.1 | Воздух |
| 5 | 20 | 2.8 | 300 | 0.09 | Воздух |

#### Задания:
1. Рассчитайте параметры во всех точках цикла
2. Определите подведённую и отведённую теплоту
3. Вычислите работу цикла и термический КПД
4. Постройте цикл в p-v координатах
5. Постройте цикл в T-s координатах

**Расчётные формулы:**
$$p_2 = p_1 \varepsilon^k$$
$$T_2 = T_1 \varepsilon^{k-1}$$
$$p_3 = p_2$$
$$T_3 = T_2 \rho$$
$$p_4 = p_3 \left(\frac{\rho}{\varepsilon}\right)^k$$
$$T_4 = T_3 \left(\frac{\rho}{\varepsilon}\right)^{k-1}$$

$$q_1 = c_p(T_3 - T_2)$$
$$q_2 = c_v(T_4 - T_1)$$
$$l = q_1 - q_2$$
$$\eta_t = 1 - \frac{1}{\varepsilon^{k-1}} \cdot \frac{\rho^k - 1}{k(\rho - 1)}$$
