- Элементный состав
- Стехиометрический коэффициент

### Часть 1. Влияние числа цилиндров

#### Теоретические основы
**Мощность двигателя:**
$$N_e = \frac{p_e \cdot V_h \cdot n \cdot i}{30 \tau}$$

где:
- $p_e$ — среднее эффективное давление, МПа
- $V_h$ — рабочий объём одного цилиндра, л
- $n$ — частота вращения, об/мин
- $i$ — число цилиндров
- $\tau$ — тактность (4 для 4-тактных)

**Крутящий момент:**
$$M_k = \frac{p_e \cdot V_h \cdot i}{2\pi \tau}$$

**Неравномерность хода:**
$$\delta = \frac{\omega_{max} - \omega_{min}}{\omega_{ср}}$$

Для рядных двигателей:
- 4 цилиндра: $\delta \approx 1/15$
- 6 цилиндров: $\delta \approx 1/40$
- 8 цилиндров: $\delta \approx 1/80$

### Часть 3. Влияние вида топлива

#### Характеристики топлив

| Топливо | Формула | Q_н, МДж/кг | L₀, кг возд/кг | Октановое число | Цетановое число |
|---------|---------|-------------|-----------------|------------------|-----------------|
| Бензин | C₇H₁₆ | 44.0 | 14.7 | 92-98 | - |
| Дизтопливо | C₁₄H₃₀ | 42.5 | 14.5 | - | 45-55 |
| Метанол | CH₃OH | 19.7 | 6.5 | 110 | - |
| Этанол | C₂H₅OH | 26.8 | 9.0 | 108 | - |
| Пропан | C₃H₈ | 46.4 | 15.7 | 105 | - |
| Метан | CH₄ | 50.0 | 17.2 | 120 | - |
| Водород | H₂ | 120.0 | 34.5 | >130 | - |

#### Теоретическая основа
**Количество воздуха для сгорания:**
$$L_0 = \frac{1}{0.23} \cdot \frac{8}{3} \cdot \frac{C + H - O}{100}$$

**Теплота сгорания смеси:**
$$Q_{см} = \frac{Q_н}{1 + \alpha L_0}$$

где $\alpha$ — коэффициент избытка воздуха

**Мощность при одинаковом рабочем объёме:**
$$\frac{N_1}{N_2} = \frac{Q_{см1}}{Q_{см2}}$$

### Часть 4. Влияние степени сжатия и состава смеси

#### Влияние степени сжатия на КПД
Для бензинового двигателя:
$$\eta_t = 1 - \frac{1}{\varepsilon^{k-1}}$$

| ε | 6 | 7 | 8 | 9 | 10 | 11 | 12 |
|---|----|----|----|----|----|----|----|
| η_t | | | | | | | |

**Постройте график** $\eta_t = f(\varepsilon)$

#### Ограничения по детонации
- Для бензина АИ-92: max ε ≈ 9-10
- Для бензина АИ-95: max ε ≈ 10-11
- Для бензина АИ-98: max ε ≈ 11-12
- Для газа: max ε ≈ 12-14
- Для спиртов: max ε ≈ 14-16

#### Влияние состава смеси (α)
| α | Характер смеси | Мощность | Экономичность | Особенности |
|---|----------------|----------|---------------|-------------|
| < 0.8 | Богатая | Падает | Плохая | Копоть, перегрев |
| 0.8-0.9 | Обогащённая | Макс | Средняя | Режим макс. мощности |
| 0.9-1.0 | Околостехиом. | Высокая | Хорошая | Нейтрализатор |
| 1.0-1.1 | Стехиометрическая | Средняя | Лучшая | Эконом-режим |
| 1.1-1.2 | Обеднённая | Снижена | Хорошая | Риск пропусков |
| > 1.2 | Бедная | Низкая | Плохая | Неустойчивая работа |
