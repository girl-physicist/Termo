# PV-диаграммы политропных процессов

## 📊 Семейство политроп на p-v диаграмме

### Общий вид
p  
↑  
│ ╱  
│ ╱ n=0 (изобара)  
│ ╱  
│ ╱  
│╱  
│───────── n=1 (изотерма)  
│╲  
│ ╲ n=k (адиабата)  
│ ╲  
│ ╲  
│ ╲ n=∞ (изохора)  
│ ╲  
└─────────→ v


### Характерные области
1. **n < 1** — процессы с подводом тепла
2. **1 < n < k** — процессы сжатия с охлаждением
3. **n > k** — процессы сжатия с нагревом

## 📐 Построение политропы в координатах p-v

### Исходные данные
Для построения необходимо знать:
- Начальные параметры ($p_1$, $v_1$, $T_1$)
- Показатель политропы ($n$)
- Конечный параметр (например, $v_2$ или $p_2$)

### Алгоритм построения
1. Задаём ряд значений объёма от $v_1$ до $v_2$
2. Для каждого $v$ вычисляем давление:
   $$p = p_1 \left(\frac{v_1}{v}\right)^n$$
3. Наносим точки на график
4. Соединяем точки плавной кривой

### Пример расчёта
**Дано:** Воздух, $p_1 = 0.1$ МПа, $v_1 = 0.8$ м³/кг, $n = 1.2$, $v_2 = 0.2$ м³/кг

| v, м³/кг | p, МПа | Точка |
|----------|--------|-------|
| 0.8 | 0.100 | 1 |
| 0.6 | 0.141 | |
| 0.4 | 0.229 | |
| 0.2 | 0.529 | 2 |

0.3┤ ╲

## 🔄 Сравнение с другими процессами

### Положение политроп относительно друг друга
При одинаковых начальных условиях:
- **Круче всех** идёт адиабата (n = k)
- **Более пологая** — изотерма (n = 1)
- **Промежуточное положение** — политропа с 1 < n < k
