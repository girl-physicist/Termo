# Расчёт параметров политропного процесса

## 📐 Калькулятор политропного процесса

### Исходные данные
| Параметр | Значение | Ед. изм. |
|----------|----------|----------|
| Начальное давление $p_1$ | | Па |
| Начальная температура $T_1$ | | К |
| Начальный объём $V_1$ | | м³ |
| Масса газа $m$ | | кг |
| Показатель политропы $n$ | | - |
| Газовая постоянная $R$ | | Дж/(кг·К) |
| $c_v$ | | Дж/(кг·К) |

### Расчётные формулы

#### 1. Связь параметров
$$\frac{T_2}{T_1} = \left(\frac{V_1}{V_2}\right)^{n-1}$$
$$\frac{p_2}{p_1} = \left(\frac{V_1}{V_2}\right)^{n}$$

#### 2. Работа процесса
$$L = \frac{mR}{n-1}(T_1 - T_2)$$

#### 3. Теплота процесса
$$Q = m c_v \frac{n-k}{n-1} (T_2 - T_1)$$

#### 4. Изменение энтропии
$$\Delta S = m c_v \frac{n-k}{n-1} \ln\frac{T_2}{T_1}$$

## 📊 Примеры расчётов

### Пример 1. Сжатие воздуха в компрессоре
**Дано:**
- $p_1 = 0.1$ МПа
- $T_1 = 300$ К
- $V_1 = 1$ м³
- $n = 1.3$
- $V_2 = 0.2$ м³
- Воздух ($R = 287$ Дж/кг·К, $c_v = 718$ Дж/кг·К, $k = 1.4$)

**Решение:**
1. Конечное давление:
   $$p_2 = p_1 \left(\frac{V_1}{V_2}\right)^n = 0.1 \times \left(\frac{1}{0.2}\right)^{1.3} = 0.1 \times 5^{1.3} = 0.1 \times 8.1 = 0.81 \text{ МПа}$$

2. Конечная температура:
   $$T_2 = T_1 \left(\frac{V_1}{V_2}\right)^{n-1} = 300 \times 5^{0.3} = 300 \times 1.62 = 486 \text{ К}$$

3. Масса газа:
   $$m = \frac{p_1 V_1}{R T_1} = \frac{0.1 \times 10^6 \times 1}{287 \times 300} = \frac{100000}{86100} = 1.16 \text{ кг}$$

4. Работа сжатия:
   $$L = \frac{mR}{n-1}(T_1 - T_2) = \frac{1.16 \times 287}{0.3}(300 - 486) = \frac{333}{0.3} \times (-186) = 1110 \times (-186) = -206460 \text{ Дж} \approx -206 \text{ кДж}$$
