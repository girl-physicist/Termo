# Общая методика расчёта термодинамических циклов ДВС

## 📋 Исходные данные

### Для всех типов циклов
| Параметр | Обозначение | Размерность |
|----------|-------------|-------------|
| Рабочее тело | - | - |
| Начальное давление | $p_1$ | Па (МПа) |
| Начальная температура | $T_1$ | К |
| Степень сжатия | $\varepsilon = V_1/V_2$ | - |
| Показатель адиабаты | $k = c_p/c_v$ | - |
| Газовая постоянная | $R$ | Дж/(кг·К) |
| Теплоёмкость | $c_v$, $c_p$ | Дж/(кг·К) |

### Для цикла Отто (дополнительно)
| Параметр | Обозначение | Размерность |
|----------|-------------|-------------|
| Степень повышения давления | $\lambda = p_3/p_2$ | - |

### Для цикла Дизеля (дополнительно)
| Параметр | Обозначение | Размерность |
|----------|-------------|-------------|
| Степень предварительного расширения | $\rho = V_3/V_2$ | - |

### Для цикла Тринклера (дополнительно)
| Параметр | Обозначение | Размерность |
|----------|-------------|-------------|
| Степень повышения давления | $\lambda = p_3/p_2$ | - |
| Степень предварительного расширения | $\rho = V_3/V_2$ | - |

## 📝 Последовательность расчёта

### Шаг 1. Расчёт параметров в точке 2 (конец сжатия)

**Давление:**
$$p_2 = p_1 \cdot \varepsilon^k$$

**Температура:**
$$T_2 = T_1 \cdot \varepsilon^{k-1}$$

**Удельный объём:**
$$v_2 = \frac{v_1}{\varepsilon} = \frac{RT_1}{p_1 \varepsilon}$$

### Шаг 2. Расчёт параметров в точке 3 (конец подвода тепла)

#### Для цикла Отто:
$$p_3 = p_2 \cdot \lambda$$
$$T_3 = T_2 \cdot \lambda$$
$$v_3 = v_2$$

#### Для цикла Дизеля:
$$p_3 = p_2$$
$$T_3 = T_2 \cdot \rho$$
$$v_3 = v_2 \cdot \rho$$

#### Для цикла Тринклера:
$$p_3 = p_2 \cdot \lambda$$
$$T_{3'} = T_2 \cdot \lambda$$
$$v_{3'} = v_2$$
$$T_3 = T_{3'} \cdot \rho = T_2 \cdot \lambda \cdot \rho$$
$$v_3 = v_2 \cdot \rho$$
$$p_3 = p_{3'} = p_2 \cdot \lambda$$

### Шаг 3. Расчёт параметров в точке 4 (конец расширения)

#### Для цикла Отто:
$$p_4 = p_3 \cdot \left(\frac{1}{\varepsilon}\right)^k = p_1 \cdot \lambda$$
$$T_4 = T_3 \cdot \left(\frac{1}{\varepsilon}\right)^{k-1} = T_1 \cdot \lambda$$
$$v_4 = v_1$$

#### Для цикла Дизеля:
$$p_4 = p_3 \cdot \left(\frac{\rho}{\varepsilon}\right)^k$$
$$T_4 = T_3 \cdot \left(\frac{\rho}{\varepsilon}\right)^{k-1}$$
$$v_4 = v_1$$

#### Для цикла Тринклера:
$$p_4 = p_3 \cdot \left(\frac{\rho}{\varepsilon}\right)^k = p_1 \cdot \lambda \cdot \left(\frac{\rho}{\varepsilon}\right)^k$$
$$T_4 = T_3 \cdot \left(\frac{\rho}{\varepsilon}\right)^{k-1} = T_1 \cdot \lambda \cdot \rho \cdot \left(\frac{\rho}{\varepsilon}\right)^{k-1}$$
$$v_4 = v_1$$

### Шаг 4. Расчёт энергетических характеристик

#### Подведённая теплота

**Цикл Отто:**
$$q_1 = c_v (T_3 - T_2)$$

**Цикл Дизеля:**
$$q_1 = c_p (T_3 - T_2)$$

**Цикл Тринклера:**
$$q_1 = c_v (T_{3'} - T_2) + c_p (T_3 - T_{3'}) = c_v T_2(\lambda - 1) + c_p T_2 \lambda (\rho - 1)$$

#### Отведённая теплота (для всех циклов)
$$q_2 = c_v (T_4 - T_1)$$

#### Работа цикла
$$l = q_1 - q_2$$

#### Термический КПД

**Цикл Отто:**
$$\eta_t = 1 - \frac{1}{\varepsilon^{k-1}}$$

**Цикл Дизеля:**
$$\eta_t = 1 - \frac{1}{\varepsilon^{k-1}} \cdot \frac{\rho^k - 1}{k(\rho - 1)}$$

**Цикл Тринклера:**
$$\eta_t = 1 - \frac{1}{\varepsilon^{k-1}} \cdot \frac{\lambda \rho^k - 1}{(\lambda - 1) + k\lambda(\rho - 1)}$$

### Шаг 5. Расчёт среднего индикаторного давления

$$p_i = \frac{l}{v_1 - v_2} = \frac{l}{v_1(1 - 1/\varepsilon)}$$

Для цикла Отто:
$$p_i = \frac{p_1}{\varepsilon - 1} \left[\lambda(\varepsilon^{k-1} - 1) - (\varepsilon^{k-1} - 1)\right]$$

## ✅ Чек-лист проверки расчёта

- Все единицы измерения согласованы (Па или МПа?)
    
- Температура в Кельвинах, а не в Цельсиях
    
- Правильно выбран тип цикла
    
- Проверено условие p₄ > p₁ (должно выполняться)
    
- КПД < КПД цикла Карно при данных температурах
    
- Работа цикла положительна
